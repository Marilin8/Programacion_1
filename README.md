# Programacion_1
En este repositorio subiremos las practicas realizadas en clase del curso de programación 1.

Ejercicios en Dev C++, haciendo uso de las estructuras repetitivas (FOR, WHILE, DO WHILE), vectores y matrices.

1. Matrices: En el primer ejemplo de Matrices ingresamos los numeros del 0 al 15, la primera matriz mostrará esos números de forma ordenada. La segunda matríz es la transpuesta de la primera, mostrará los números del 0 al 15 pero en diferente orden.
2. Vectores: Es un tipo de array de una sola dimensión, en este ejemplo el programa nos mostrará los números impares que existen entre 1 y 100.
3. Etructura repetitiva FOR: Se utiliza cuando el usuario quiere repetir un determinado número de veces ciertas sentencias mientras se mantenga el cumplimiento de una condición. En este ejemplo el programa mostrará el total de la suma de todos los números impares entre 1 y 100.
4. Estructura repetitiva WHILE: Tiene asociado un bloque de sentencias que se ejecutarán secuencialmente mientras la condición de entrada sea cierta. En este ejemplo el programa mostrará la palabra "Hola" 100 veces.
5. Estructura repetitiva DO-WHILE: Al contrario del bucle While, esta estructura evalúa la condición al final del bucle. En el ejemplo mostrado, el programa le mostrará una serie de instrucciones al usuario donde le pedirá que ingrese ciertos datos para ir evaluando y de esta manera poder dar un resultado al finalizar el programa.
